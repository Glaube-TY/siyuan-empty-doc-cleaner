# 插件简介

- 该插件可以帮助你自动检索思源笔记工作空间中的所有空文档，并生成一个包含所有空文档的列表。

- 点击列表中的文档名称，即可预览文档内容。（确定是否为空，以免误删）

- 选择你想删除的文档（支持多选），该插件会自动删除选中的文档。
